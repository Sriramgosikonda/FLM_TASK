# Companies Directory – Frontend Application

This project was developed as part of the Frontlines Media technical assessment for the **Frontend Developer role**.  
It is a fully responsive React-based application that displays a list of companies with filtering, searching, and sorting features.

---

##  Features

###  Core Features Implemented
- Responsive UI built using **React.js** and **Tailwind CSS**
- Company list displayed in card/table layout
- Filters for:
  - Company Name
  - Location
  - Industry
- Search functionality for instant lookups
- Proper loading and error handling
- State management using **React Hooks**

###  Bonus Features Added
- Pagination for browsing large datasets
- Sorting (A–Z / Z–A) by company name
- Enhanced UI using Tailwind components
- Live deployment on Vercel

---

##  Tech Stack
- **Frontend:** React.js, Vite/CRA, Tailwind CSS  
- **State Management:** React Hooks  
- **API:** JSON Server / Mock API / Static JSON file  
- **Deployment:** Vercel / Netlify
